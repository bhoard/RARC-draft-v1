# Richmond Area RC Brand Spec

Derived from the provided Richmond Flying Club reference and the requested Spotify-derived dark palette with red replacing green.

## Tokens

- `--bg`: `oklch(16.7% 0.031 260)` near-black navy canvas
- `--surface`: `oklch(21.5% 0.024 260)` dark card/surface
- `--fg`: `oklch(100% 0 0)` primary white text
- `--muted`: `oklch(74% 0.012 255)` silver secondary text
- `--border`: `oklch(35% 0.018 255)` low-contrast dark border
- `--accent`: `oklch(50% 0.19 28)` club red for CTAs, active states, and focus only

## Type

- Display: `Manrope`, then system sans-serif fallbacks
- Body: `Manrope`, then system sans-serif fallbacks
- Mono: `ui-monospace`, `SFMono-Regular`, `Menlo`, `Consolas`, monospace

## Layout Posture

- Black logo/navigation band because the supplied logo will sit on a black background.
- Large photography-led hero with dark overlay and concise club positioning.
- Rounded pill CTAs and compact dark surfaces inspired by the active system.
- Red is functional, not decorative: primary action, selected state, focus ring.
- Photography placeholders should be labeled as CC/open-license replacement slots until final club images are available.
